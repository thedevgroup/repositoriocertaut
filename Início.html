Como instalar certificados raiz?

Internet Explorer/EDGE, Chrome e Opera (Windows):
Baixe o certificado raiz ca.p12 (formato .p12 PKCS#12)
Abra o arquivo ca.p12
O menu de Importação de Certificados do Windows deverá abrir
Complete a importação
Na página da senha: deixe em branco e aperte Next (ou Próximo)
Reinicie seu navegador

Nota: IE/EDGE, Chrome e Opera usam a Loja de Certificados do Windows. Estes navegadores carregam automáticamente os certificados raiz da Loja de Certificados do Windows.

Firefox:
Baixe o certificado raiz cacert-info.cer (formato .cer PEM)
O menu de Importação de Certificados do Firefox deve abrir automaticamente
Marque (habilitar) "Confiar neste CA para identificar websites"
Aperte OK para instalar o certificado raiz.


Android:
Baixe o certificado raiz cacert-info.cer (formato .cer PEM)
Usando um gerenciador de arquivos clique no arquivo.
Digite um nome para ele
Aperte OK para instalar o certificado raiz.
